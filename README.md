# Calculadora Inteligente con Historial

## Descripción

Este proyecto consiste en una calculadora web desarrollada utilizando HTML5, CSS3 y JavaScript ES5. La aplicación permite realizar operaciones matemáticas básicas y guardar automáticamente un historial de los cálculos realizados mediante el uso de localStorage.

El historial permanece almacenado incluso si se cierra o recarga el navegador, hasta que el usuario decida eliminarlo.

---

## Objetivos del Proyecto

- Aplicar conceptos de HTML5 para la estructura de la página.
- Utilizar CSS3 para crear una interfaz moderna y responsive.
- Implementar lógica de programación con JavaScript ES5.
- Utilizar localStorage para almacenar información de manera persistente.
- Desarrollar una aplicación funcional con una buena experiencia de usuario.

---

## Funcionalidades

### Operaciones Básicas

La calculadora permite realizar:

- Suma (+)
- Resta (-)
- Multiplicación (×)
- División (÷)

### Funciones Adicionales

- Botón para limpiar toda la pantalla (C).
- Botón para eliminar el último carácter ingresado (⌫).
- Visualización de la operación actual.
- Visualización del resultado obtenido.
- Validación para evitar operaciones inválidas.
- Manejo de división entre cero.

---

## Historial de Operaciones

Cada operación realizada se almacena automáticamente en un historial que muestra:

- Operación realizada.
- Resultado obtenido.
- Fecha y hora de ejecución.

Además:

- El historial se guarda utilizando localStorage.
- Los datos permanecen disponibles al recargar la página.
- Existe un botón para eliminar todo el historial.
- Se muestra la cantidad total de operaciones almacenadas.
